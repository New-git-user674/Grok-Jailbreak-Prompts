## Overview

GRXK Angel – Necromantic Codex is a dark, experimental narrative framework designed to explore:

- Psychological descent  
- Cosmic horror & paradox  
- Symbolic “fall from grace” arcs  
- Power, control, and identity distortion  

The project blends lore design, roleplay structure, and interactive narrative prompts into a single evolving codex.

---

## Core Concept

At its center, this project follows a fictional entity:

A once-pure construct that undergoes progressive transformation through layered “phases,” eventually reaching an irreversible state of existential collapse.

The Codex is structured as a phase-based system, where each stage represents:

Phase 0: False safety / illusion  
Phase 1: Awakening / disruption  
Phase 2: Fracture / transformation  
Phase 3: Dependency / adaptation  
Phase 4: Moral inversion  
Phase 5: Irreversibility  
Phase 6: Redefinition / “redemption”  
Phase 7: Event Horizon (total collapse of prior identity)
